# 觀測者月亮視角及角度變化實作計畫

本計畫旨在網頁互動模擬器中新增「觀測者地平線視角」以及「地球自轉/地方時間控制」。這將使學生能夠直觀地理解在地球上不同時間、不同月相下，月亮在天空中的實際仰角、方位以及月面傾角的變化。

## 使用者審查要求

> [!IMPORTANT]
> 為了實現完整的觀測者視角，我們需要同時控制：
> 1. **月球公轉軌道位置**（決定農曆日期與月相變化，如初一至三十）。
> 2. **地球自轉位置/觀測者地方時間**（決定一天中的時間，如清晨、正午、黃昏、午夜）。
> 
> 我們將新增一個「觀測者時間 (地球自轉)」滑桿與快速時間預設按鈕，以便使用者手動調整或播放地球自轉。

> [!TIP]
> 為了讓視覺效果更加引人入勝且具教育意義：
> - **軌道模擬器**中將繪製一個代表「觀測者」的小圖示（🔭/🧍）以及其「地平線與可見天空範疇」（以綠色半圓扇形標示）。
> - **新增地平線視角畫布**（Horizon View Canvas），根據觀測者時間動態調整天空顏色（白天的藍天、黃昏與黎明的晚霞、夜晚帶有星星的星空），並繪製高空中的太陽與月球，且月球會根據太陽的相對位置顯示正確的月相與傾角。

## 預計修改內容

### [網頁前端組件]

#### [MODIFY] [index.html](file:///c:/Earth-Science-1/index.html)

1. **HTML 結構調整**：
   - 在第一個分頁（`#tab0`）的右側欄（`.moon-preview`）中，新增一個「觀測者地平線視角」畫布（`#horizonCanvas`，寬 260px，高 150px）。
   - 在控制區域（`.controls`）中新增「地球自轉（地方時間）」控制項：
     - 一個自轉角度/時間滑桿（00:00 - 24:00）。
     - 四個快速切換按鈕：正午 (12:00)、黃昏 (18:00)、午夜 (24:00)、清晨 (06:00)。
     - 一個「地球自轉播放」切換鈕，可以讓地球自己轉動，或與月球公轉同步轉動（如 1 天自轉 360°，月球公轉約 12.2°）。

2. **CSS 樣式調整**：
   - 優化 `.moon-preview` 內部的佈局，使其能整齊容納「地球看到的月相」與「觀測者地平線視角」兩個畫布。
   - 增加觀測者控制面板的樣式，維持既有的深藍色高質感科技風（Glassmorphism 玻璃擬物化風格、陰影與漸層）。

3. **JavaScript 邏輯新增與修改**：
   - **新增變數**：`earthAngle` (地球自轉角度，0° - 360°，對應 12:00 - 12:00 或自訂時間點)。
   - **繪製軌道模擬器 (`drawOrbitSimulator`)**：
     - 在地球（中心）的外圈繪製觀測者位置（以 `earthAngle` 決定角度）。
     - 繪製觀測者的地平線（切線）與可見天空範圍（綠色半透明 180° 扇形）。
     - 當月球落在該綠色扇形內時，代表月球「已升起」，否則為「在地平線下（不可見）」。
   - **新增地平線視角畫布繪製 (`drawHorizonView`)**：
     - 取得觀測者相對於太陽的角度 $\phi_{Sun} = -earthAngle$。
     - 取得觀測者相對於月球的角度 $\phi_{Moon} = moonAngle - earthAngle$。
     - 判斷太陽和月亮是否在地平線上（角度差在 $\pm 90°$ 內）。
     - 依據太陽的高度繪製天空背景色：
       - 日正當中：蔚藍色。
       - 接近地平線：橙紅/金黃/紫色的夕陽或晨曦。
       - 地平線下：繁星點點的夜空。
     - 在畫布上投影太陽與月亮位置：
       - 方位角以左右位置表示（左為東 E，右為西 W，中間為南 S/天頂）。
       - 仰角以垂直高度表示（從地平線到畫布頂端）。
     - 繪製月亮時，依據太陽在畫布上的相對方向，旋轉月亮亮面，使亮面精確指向太陽！
   - **更新資訊面板**：
     - 顯示精確的地方時間（例如：`14:30`）。
     - 顯示月亮的仰角與方位。
     - 顯示月亮當前狀態（例如：「地平線上（可見）」或「地平線下（不可見）」）。

## 驗證計劃

### 手動測試步驟
1. **載入頁面**：開啟 `index.html`，檢查右側欄是否新增了「觀測者地平線視角」畫布以及下方的「地球自轉/地方時間」控制項。
2. **調整地方時間**：
   - 拖動地方時間滑桿或點擊快速按鈕（如黃昏 18:00）。
   - 觀察軌道模擬器中的綠色觀測者視角扇形是否隨之旋轉。
   - 觀察地平線畫布的天空顏色是否流暢過渡（中午藍天 $\to$ 黃昏橘紅 $\to$ 午夜黑夜）。
3. **驗證月亮可見性與位置**：
   - 將月亮置於上弦月位置（`moonAngle = 90°`）。
   - 點擊「正午 12:00」：此時月亮應在東方地平線剛升起（仰角 $\approx 0°$，方位為東），天空為藍色。
   - 點擊「黃昏 18:00」：此時月亮應在天頂最高處（仰角 $\approx 90°$），天空呈黃昏色。
   - 點擊「午夜 24:00」：此時月亮應在西方地平線即將落下（仰角 $\approx 0°$，方位為西），天空為黑夜。
   - 觀察地平線畫布中月亮位置、仰角標示與亮面朝向（右半邊亮且朝向太陽方向）是否完全正確。
4. **自轉與公轉連動播放**：
   - 啟動播放，觀察地球與月亮同時運動時，地平線畫布中日月升落的動態效果。
